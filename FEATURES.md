# 🎮 Portfolio Features

## ✨ Visual Effects

### 🌟 Particle Background
- Animated pixel particles floating in the background
- Pulsing opacity for dynamic effect
- Optimized canvas rendering

### 🖱️ Cursor Trail
- Interactive pixel trail that follows your mouse
- Particles with physics (gravity, fade out)
- Cyan/blue color scheme matching the theme

### 📺 CRT Monitor Effects
- **Scanlines**: Horizontal lines simulating old CRT monitors
- **Flicker**: Subtle screen flicker effect
- **Vignette**: Dark edges for depth and focus

### 📊 Scroll Progress Indicator
- Top progress bar showing scroll position
- Gradient color (primary → accent → destructive)
- Percentage indicator in bottom-right corner

## 🎨 CSS Animations

### Glitch Effect
- Random glitch animation on hover
- RGB color split effect
- Cyberpunk aesthetic

### Pixel Border Hover
- 3D depth effect on hover
- Color transition to primary color
- Smooth cubic-bezier transitions

### Floating Animation
- Smooth up/down movement
- Used on icons and elements
- Staggered delays for variety

### Neon Glow
- Pulsing glow effect
- Multiple shadow layers
- Perfect for headings and buttons

### Additional Animations
- `slide-in`: Fade in from bottom
- `fade-in`: Simple opacity transition
- `pixel-shimmer`: Brightness pulse
- `cursor-blink`: Typewriter cursor effect

## 🎯 User Experience

### Smooth Scroll
- Native smooth scrolling behavior
- Anchors animate smoothly

### Responsive Design
- Mobile-first approach
- Pixel-perfect on all screen sizes
- Touch-friendly interactions

### Performance Optimized
- Canvas animations with requestAnimationFrame
- Efficient particle system
- Minimal re-renders

## 🎨 Design System

### Color Palette
- **Primary**: Blue (`hsl(212 100% 67%)`)
- **Accent**: Green (`hsl(142 70% 60%)`)
- **Destructive**: Red (`hsl(4 90% 64%)`)
- **Background**: Dark blue (`hsl(222 47% 11%)`)

### Typography
- **Headings**: Press Start 2P (pixel font)
- **Body**: JetBrains Mono (monospace)
- Pixelated rendering for retro feel

### Components
- Pixel borders with drop shadows
- Card-based layout
- Consistent spacing and rhythm

## 🚀 How to Use

1. **Cursor Trail**: Move your mouse to see the pixel trail
2. **Scroll Progress**: Scroll down to see the progress bar fill
3. **Hover Effects**: Hover over cards and buttons for animations
4. **CRT Effect**: Enjoy the retro monitor aesthetic throughout

## 🛠️ Technical Stack
- React 19
- TypeScript
- Tailwind CSS v3
- Vite
- Canvas API for effects
